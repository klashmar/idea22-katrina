What Can JavaScript Do?

With jQuery, you can hide and show HTML elements with the hide() and show() methods. JavaScript methods are the actions that can be performed on objects. In JavaScript, almost "everything" is an object. If you understand objects, you understand JavaScript.

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

The optional speed parameter specifies the speed of the hiding/showing, and can take the following values: "slow", "fast", or milliseconds.
